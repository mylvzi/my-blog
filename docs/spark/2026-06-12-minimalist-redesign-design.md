# 博客极简风格重设计方案

日期: 2026-06-12 | 状态: 实施中

## 目标

简化博客视觉呈现，从"花里胡哨"转向 Claude Code 网页版式的**温暖人本主义极简**风格。保留 Stellar 主题的 HTML 结构，通过重写 CSS + 精简 JS 实现。

## 设计原则

| 维度 | 当前 | 目标 |
|------|------|------|
| 配色 | 高饱和蓝/青/琥珀/玫瑰多色 | 暖白 `#faf8f5` + 深棕 `#3d3629` + 低饱和大地色点缀 |
| 背景 | 光标跟踪径向渐变 + 网格 + 对角线 | 纯色暖白，无纹理 |
| 阴影 | 大扩散阴影 (14-34px) | 极细微阴影 (0 1px 3px) 或无阴影 |
| 玻璃拟态 | 导航/卡片/侧边栏/播放器全覆盖 | 完全去除 |
| 动画 | 启动画面、悬停上移、强调条淡入 | 仅保留基础过渡 |
| 卡片 | 彩色强调条 + 图标 + 徽章 + 芯片 + 阅读时间 | 干净标题 + 摘要，无装饰 |
| 排版 | 标准间距 | 行高 1.8，段落间距加大，页宽收窄 |

## 变更清单

### 删除
- `source/js/blog-effects.js` — 启动动画、光标跟踪、进度条
- `source/js/blog-runtime.js` — 实时计时器
- `scripts/footer-stats.js` — 页脚统计注入
- `scripts/polish-homepage.js` — 首页卡片装饰
- `scripts/heatmap-helper.js` — 热力图组件

### 修改
- `source/css/theme-toggle.css` — 完全重写，新极简风格
- `source/js/theme-toggle.js` — 精简
- `source/_data/widgets.yml` — 简化侧边栏
- `_config.stellar.yml` — 移除已删除脚本引用、调整配置
- `scripts/reading-notes.js` — 移除热力图引用，改写为纯列表
- `scripts/practice-log.js` — 移除热力图引用，改写为纯列表

### 保留
- `source/js/music-player.js` — 用户明确要求
- Utterances 评论
- Font Awesome 6.4.2 CDN

## 验收标准

1. 首页无启动动画、光标跟踪、彩色卡片装饰
2. 文章卡片为简洁标题+摘要，无图标/徽章/芯片/彩色条
3. 无阅读进度条
4. 无页脚统计和实时计时器
5. 背景为纯色暖白，无渐变网格
6. 无玻璃拟态效果
7. 音乐播放器正常工作
8. 暗色/亮色切换正常
9. 阅读笔记和实践日志页面正常（无热力图）
10. 构建无报错
